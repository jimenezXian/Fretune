# Fretune

A cross-platform guitar tuning app built with Expo and React Native. Tune your instruments on mobile and web

## Features

- Real-time guitar tuning
- Store and save instruments with custom tunings
- Cross-compatible across iOS, Android, web
- Persistent local storage with MMKV
- Lightweight state management with Zustand

## Getting Started

1. Install dependencies

   ```bash
   bun install
   ```

2. Start the app

   ```bash
   bun start
   ```

   From there you can open the app on:
   - Android emulator
   - iOS simulator
   - web
